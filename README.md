# 🛍️ DVL E-Commerce Frontend

A clean, responsive, and fully functional **E-Commerce Frontend** built using **HTML, CSS, and JavaScript**.  
This is a **real-world UI implementation** designed to work seamlessly with the backend API and deliver a smooth shopping experience.

---

## 🚀 Overview

This frontend powers the **DVL (clothing brand) E-Commerce platform**, providing users with an interactive and user-friendly interface to browse products, manage carts, and place orders.

It focuses on **performance, simplicity, and real-world usability** without relying on frameworks.

---

## 🔄 Application Flow

1. User lands on the homepage  
2. Browses clothing products  
3. Views product details  
4. Adds items to cart  
5. Logs in / signs up (OTP-based authentication)  
6. Proceeds to checkout  
7. Adds shipping address  
8. Places order  
9. Views profile and order details

## ⚙️ Setup & Installation

1. Clone the repository

```
git clone https://github.com/imdevvx/Ecommerce_frontend.git
cd Ecommerce_frontend
```

2. Run the project

Simply open:

```
index.html
```

Or use Live Server in VS Code for a better experience.

## 🔗 Backend Integration

This frontend is designed to work with the E-Commerce backend API.

Make sure the backend server is running and API URLs are correctly configured inside:

```
/js/config/config.js
```
